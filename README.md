## 🚀 Deploying to GitHub Pages

1. **Create a new GitHub repository** (e.g., `kaustubh-portfolio`)
2. **Push all files** to the `main` branch:
   ```bash
   git init
   git add .
   git commit -m "Initial commit — portfolio website"
   git remote add origin https://github.com/YOUR_USERNAME/kaustubh-portfolio.git
   git push -u origin main
   ```
3. **Enable GitHub Pages:**
   - Go to your repo → **Settings** → **Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` → `/ (root)`
   - Click **Save**
4. Your site will be live at `https://YOUR_USERNAME.github.io/kaustubh-portfolio/`
